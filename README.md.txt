# 🧾 QR Code Generator (Python)

This is a simple Python program that generates QR codes from user input (text or URLs) using the `qrcode` library.

## 💡 Features
- Generate a QR code from any text or URL
- Saves the QR code image with a custom filename
- Clean and beginner-friendly code


## 🛠️ Requirements
Install the required Python library:
```bash
pip install qrcode
